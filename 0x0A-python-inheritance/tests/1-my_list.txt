Using ''print_sorted''
-------------------
This is an example text file . First import
''print_sorted'' from the ''1-my_list'' module:
        >>> MyList = __import__('1-my_list').MyList
        >>> my_list = MyList([1, 4, 5, 3, 2])
        >>> my_list.print_sorted()
	>>> [1, 2, 3, 4, 5]
	>>> my_list = MyList([1, 2, 3, 2, 3, 5])
	>>> [1, 2, 2, 3, 3, 5]
